# 🌟 Znajdź Swoją Grupę - Clustering App

Interaktywna aplikacja do znajdowania osób o podobnych zainteresowaniach używająca Machine Learning (K-Means Clustering).

## 🚀 Features
- Predykcja klastra na podstawie preferencji użytkownika
- Interaktywne wizualizacje (Plotly)
- Glamour design z gradientami
- Analiza demograficzna klastrów

## 🛠️ Tech Stack
- Streamlit
- PyCaret
- Plotly
- Pandas

## 📦 Instalacja lokalna

```bash
pip install -r requirements.txt
streamlit run app.py
